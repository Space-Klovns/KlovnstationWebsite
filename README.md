Klovnstation 14 _(KS14)_ is a fork of [Space Station 14](https://github.com/space-wizards/space-station-14).
[Our discord server.](https://discord.gg/rSjZ48VnMx)

This is the website for Klovnstation 14. It is a static website designed to be hosted on Github Pages.

## How to contribute
If you wish to contribute to the website, you will need to:
- Download and install Hugo onto your computer. We recommend you use the winget method as it is very simple.
- Fork and then clone this repository
- Make the changes you want (we recommend you copy from the existing files for ease of use)
- Verify that the website works (by using hugo server)
- Make a pull request to our repository

Do know that changes not approved in advance by Space Klovns are less likely to be accepted. We recommend you join our discord server and consult whatever changes you wish to perform with us.
